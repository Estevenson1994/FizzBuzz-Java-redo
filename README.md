# FizzBuzz in Java Using JUnit4

This is a redo of FizzBuzz using Java and the testing framework JUnit4.

## Tech

- Java
- JUnit 4 as testing framework

## How it works

```
jshell> private FizzBuzzConverter fizzBuzz = new FizzBuzzConverter();
fizzBuzz ==> FizzBuzzConverter@5c3bd550

jshell> fizzBuzz.convert(4)
$3 ==> "4"

jshell> fizzBuzz.convert(3)
$4 ==> "Fizz"

jshell> fizzBuzz.convert(5)
$5 ==> "Buzz"

jshell> fizzBuzz.convert(15)
$6 ==> "FizzBuzz"
```
